LifeOS v2 Design Spec

Complete visual and functional redesign

Visual Style: Glassmorphism

  • Frosted glass effects with backdrop-filter blur
  • Vibrant gradient backgrounds (purple/pink for main, green for finances, pink for health, orange for gym, purple for brand)
  • Semi-transparent cards with subtle borders
  • Soft shadows and depth
  • Modern, premium aesthetic

Main Dashboard

Hero Section (top):

  • Greeting with date
  • 4 key metrics in a row: Day Progress, Recovery, Streak, Goals
  • Glassmorphism card with blur effects

Timeline Section (below):

  • Vertical timeline showing day's flow
  • Completed items (green checkmark)
  • Current item (yellow, pulsing, highlighted border)
  • Upcoming items (dimmed)
  • Each item shows time, status, title, and details

Accountability Feature

Top Card (cannot dismiss):

  • Red warning banner with shake animation
  • Lists missed goal(s)
  • Textarea for explanation
  • "Submit & Continue" button (can't dismiss without submitting)
  • Rest of app dimmed until accountability resolved

Timeline Notes:

  • Missed goals show with red indicator
  • Accountability note displayed inline in timeline card
  • Shows explanation for future reference

Finances Page

  • Business cards at top with revenue/expenses/orders
  • Net worth summary below
  • Focus on income sources
  • Green gradient theme

Health Page

  • Grid layout with recovery ring and all metrics
  • Sleep, strain, RHR, HRV all visible
  • Pink gradient theme

Gym Page

  • Shows workout split (Push/Pull/Legs) with current day highlighted
  • Today's workout details below
  • Orange gradient theme

Brand Page

  • Each platform gets its own card
  • Icon, handle, follower count, weekly growth
  • Focus on per-platform tracking
  • Purple gradient theme

Navigation

  • Bottom navigation bar with glassmorphism
  • 6 tabs: Main, Finances, Brand, Health, Gym, Settings
  • Active tab highlighted

Color Palette

  • Main: Purple gradient (#667eea → #764ba2)
  • Finances: Green gradient (#10b981 → #059669)
  • Health: Pink gradient (#ec4899 → #be185d)
  • Gym: Orange gradient (#f97316 → #ea580c)
  • Brand: Purple gradient (#8b5cf6 → #7c3aed)
  • Success: #4ade80
  • Warning: #fbbf24
  • Danger: #ef4444

Typography

  • Headings: Bold, 700 weight
  • Body: Regular, 400-500 weight
  • Labels: Uppercase, small, 600 weight
  • Numbers: Bold, 700 weight

Animations

  • Page transitions: Slide in/out
  • Card entrance: Fade up
  • Current timeline item: Pulse
  • Accountability card: Shake
  • Button interactions: Scale on tap/hover
